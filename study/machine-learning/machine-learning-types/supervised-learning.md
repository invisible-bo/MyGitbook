# Supervised learning

## 1. Regression model(회귀 모델)

1. 선형 회귀\
   \- 종속 변수와 하나 이상의 독립 변수 간의 선형 관계를 모델링 하는 방법

선형관계 : 종속 변수와 독립 변수의  관계가 간단한 곱셈과 덧셈만으로 이루어져 있는것\
&#x20;                   직선으로 이루어진 표현식\
\
ex) 키, 몸무게, 나이\
키 : 종속  변수(예측하는변수)\
몸무게,  나이 : 독립 변수(예측하기 위해 사용하는 변수)



다중 선형 회귀와 단순 선형 회귀
